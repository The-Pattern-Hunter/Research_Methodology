# Research Methodology for BSc Students

This repository contains a comprehensive set of Jupyter Notebooks designed for BSc students to master the fundamentals and advanced concepts of **Research Methodology**. These resources are structured to guide students through the entire research process, from foundational theory to ethical considerations and data reporting.

### **Author Information**
* **Author:** Alok Patel, PhD (Lead developer of Pattern Hunter Philosophy)
* **Institute:** Kuchinda College, Kuchinda
* **Target Audience:** BSc Students (Research Methodology Paper)
